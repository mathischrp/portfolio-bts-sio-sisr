##  Compétences techniques mises en œuvre

- **Réseau :** Configuration TCP/IP, DHCP, NAT, pare-feu, VLANs, Cisco Packet Tracer
- **Systèmes :** Windows 10/11, Windows Server 2019, Active Directory, GPO
- **Administration :** Gestion des comptes et groupes AD, droits d'accès, partages réseau
- **Sauvegarde :** Windows Server Backup, NAS Synology, test de restauration
- **Inventaire :** OCS Inventory, recensement manuel du parc
- **Outils :** MikroTik WinBox, Cisco Packet Tracer, Microsoft Office
